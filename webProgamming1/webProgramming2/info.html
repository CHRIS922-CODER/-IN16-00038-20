HTML:HyperText Markup Language

HTML(HyperText Markup Language) is the most basic building block of the web. It defines the meaning and structure of web content. Other technologies besides HTML are generally used to describe a web page's appearance/presentation(CSS)or functionality/behavior(javascript)

"HyperText" refers to links that connect web pages to one another, either within a single website or between websites.Links are fundamental aspect of the web.By uploading content to the internet and linking it to pages created by other people, you become an active participant in the World Wide Web.
